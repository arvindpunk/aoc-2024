# Advent of Code 2024

### how to run

```bash
# To solve day N
# 1 <= N <= 25
./solve N
```

### requirements

- [clickhouse-local](https://clickhouse.com/docs/en/operations/utilities/clickhouse-local) - can be installed via nix as well, no clickhouse server required

### notes

#### naming conventions

- input: `input_dayN.txt`
- sql: `dayN.sql`
